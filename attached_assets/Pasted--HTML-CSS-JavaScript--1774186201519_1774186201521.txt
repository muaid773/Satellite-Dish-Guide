🧠 الفكرة العامة:

أريد إنشاء تطبيق ويب (HTML + CSS + JavaScript) يعمل كأداة احترافية لمساعدة المستخدم على ضبط طبق استقبال الأقمار الصناعية (Satellite Dish Alignment Tool) بطريقة بصرية تفاعلية.

📍 المدخلات:
حقل لإدخال الإحداثيات بصيغة:
15°40'01.5"N 43°57'25.8"E
أو إدخال خط العرض وخط الطول بصيغة عشرية.
اختيار القمر الصناعي من قائمة:
Nilesat 7W
Arabsat 26E
Hotbird 13E
Astra 19.2E
🛰️ عرض الخريطة:
يجب عدم استخدام خريطة تفاعلية (مثل Google Maps iframe).
يتم تحميل صورة قمر صناعي ثابتة (Static Satellite Image).
الصورة تكون:
مركزها هو الإحداثيات المدخلة.
مستوى الزوم قريب (يمثل مساحة 10–20 متر تقريبًا).
يتم وضع الصورة داخل دائرة (Circular Mask) بحيث تبدو كأنها قرص أو ساعة.
🧭 الساعة الاتجاهية (Compass UI):
فوق الصورة يتم رسم دائرة مثل الساعة (360 درجة).
يتم رسم:
خطوط كل 5 درجات.
خطوط أوضح كل 10 درجات.
علامات رئيسية:
N (0°)
E (90°)
S (180°)
W (270°)
📡 اتجاه القمر:
يتم حساب زاوية السمت (Azimuth).
يتم رسم:
خط من مركز الدائرة إلى الحافة يمثل اتجاه القمر.
أيقونة قمر (SVG) على محيط الدائرة.
تسمية القمر بجانب الأيقونة.
يجب أن يكون كل شيء:
متحرك ديناميكيًا عند تغيير الإحداثيات.
📐 الحسابات:

يجب حساب القيم التالية:

Azimuth (الزاوية الأفقية)
اتجاه الطبق بالنسبة للشمال:
0° = شمال
90° = شرق
180° = جنوب
270° = غرب
Elevation (زاوية الارتفاع)
زاوية ميل الطبق للأعلى:
0° = الأفق
90° = عمودي للأعلى
LNB Skew (ميل العدسة)
زاوية دوران رأس الاستقبال.
📊 عرض النتائج:
عرض القيم بشكل واضح:
Azimuth
Elevation
Skew
📏 مؤشر الارتفاع:
رسم شريط أفقي (مثل مسطرة).
يحتوي على:
مؤشر يتحرك حسب زاوية Elevation.
تمثيل بصري للزاوية (خط مائل).
🛰️ رسم الطبق (2D Diagram):
رسم طبق استقبال بشكل ثنائي الأبعاد (SVG).
يحتوي على:
اتجاه الطبق (خط يشير نحو القمر).
خط يمثل زاوية الارتفاع.
خط يمثل ميل LNB.
الشكل يكون رمزي مثل:
<___\
